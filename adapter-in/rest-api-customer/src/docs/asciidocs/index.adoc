== 개요
OTA 숙박 플랫폼의 Customer(고객) API 문서입니다.
고객이 숙소를 검색하고, 요금을 비교하고, 예약/취소할 수 있는 API를 제공합니다.

include::{coreDocsDir}/common.adoc[leveloffset=+1]

=== 기본 정보
|===
| 항목 | 내용
| Content-Type | `application/json`
| 인증 | (추후 적용)
|===

=== 엔드포인트 요약
|===
| 메서드 | 경로 | 설명
| GET | `/api/v1/search/properties` | 숙소 검색
| GET | `/api/v1/properties/{propertyId}/rates` | 요금 조회
| POST | `/api/v1/reservation-sessions` | 예약 세션 생성 (재고 선점)
| POST | `/api/v1/reservations` | 예약 확정
| PATCH | `/api/v1/reservations/{reservationId}/cancel` | 예약 취소
|===
