== 개요
OTA 숙박 플랫폼의 Extranet(파트너) API 문서입니다.
파트너가 숙소, 객실, 요금 정책을 등록하고 관리할 수 있는 API를 제공합니다.

include::{coreDocsDir}/common.adoc[leveloffset=+1]

=== 기본 정보
|===
| 항목 | 내용
| Base URL | `/api/v1/extranet`
| Content-Type | `application/json`
| 인증 | (추후 적용)
|===

=== 엔드포인트 요약
|===
| 메서드 | 경로 | 설명
| GET | `/properties` | 숙소 목록 조회
| GET | `/properties/{propertyId}` | 숙소 상세 조회
| POST | `/properties` | 숙소 기본정보 등록
| PUT | `/properties/{propertyId}/photos` | 숙소 사진 설정
| PUT | `/properties/{propertyId}/amenities` | 숙소 편의시설 설정
| PUT | `/properties/{propertyId}/attributes` | 숙소 속성값 설정
| POST | `/properties/{propertyId}/rooms` | 객실 유형 등록
| POST | `/properties/{propertyId}/rooms/{roomTypeId}/rate-plans` | 요금 정책 등록
| PUT | `/rate-plans/{ratePlanId}/rates` | 요금/재고 설정
|===
