## AI-9: convention-guardian — ArchUnit 규칙 보강
- **도구**: Claude Code Agent (convention-guardian 에이전트)
- **목적**: 기존 ArchUnit 규칙 9개에 신규 3개를 추가하여 컨벤션 자동 검증 강화
- **AI 제안**: 3개 신규 규칙 — Instant.now() 직접 호출 금지(테스트 가능성), Enum displayName() 필수(한국어 표시), jakarta.validation 의존 금지(도메인 순수성)
- **본인 판단**: 
  - Instant.now() 금지는 테스트 시 시간 제어를 위해 반드시 필요한 규칙. Clock 주입 패턴을 강제
  - Enum displayName()은 관리자 화면에서 Enum 값을 표시할 때 필요하므로 수용
  - jakarta.validation 금지는 Hexagonal Architecture에서 도메인 순수성 유지를 위해 수용. 검증은 adapter 레이어에서 수행
- **결과**: ArchUnit 규칙 총 12개로 강화. PO 에이전트와 병렬 실행하여 시간 절약
